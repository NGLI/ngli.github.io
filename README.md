    - `wugniu_soutseu`：苏州
- ℞ **[`NGLI/rime-wugniu_kashin`](https://github.com/NGLI/rime-wugniu_kashin)**：嘉兴吴语（五县两区）
    - `wugniu_donshian`：桐乡
    - `wugniu_haegnin`：海宁
    - `wugniu_haeye`：海盐
    - `wugniu_kashin`：嘉兴
    - `wugniu_kazoe`：嘉善
- ℞ **[`NGLI/rime-wugniu_gninpou`](https://github.com/NGLI/rime-wugniu_gninpou)**：宁波吴语
    - `wugniu_gninpou`：宁波
